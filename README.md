 🌐 Personal Portfolio Website

📌 About the Project
This is a personal portfolio website created using HTML, CSS, and JavaScript.  
It showcases my personal details, skills, and projects, and also includes a simple calculator web page.

This project helps in presenting my profile in a clean and professional way and improves my front-end development skills.

---
✨ Features

- 🏠 Home page (`index.html`)
- 🧑 Personal portfolio page (`portfolio.html`)
- 🧮 Working calculator (`calculator.html`)
- 🎨 Stylish design using CSS
- 🖼️ Profile image support (`spurthi.png`)
- 📱 Simple and easy navigation between pages

---

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (basic level for calculator)
